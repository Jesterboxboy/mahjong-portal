Еще одна версия турнирного рейтинга России по риичи маджонгу. Все еще в разработке.

Расчет рейтинга

Для каждого турнира мы расчитываем турнирный коэффициент и базовый рейтинг игрока.

И из них расчитывается дельта рейтинга:

delta = base_rank * tournament_coefficient

Базовый рейтинг игрока

Базовый рейтинг игрока зависит от позиции игрока в турнире

  • Первое место = 1000 очков
  • Середина таблицы = 0 очков
  • Последнее место = -1000 очков

Все другие места рассчитываются по формуле:

base_rank = ((number_of_players - place - middle) / (number_of_players - middle - 1)) * 1000

Турнирный коэффициент

Турнирный коэффициент базируется на количестве игроков турнира и количестве игровых сессий.

Количество игроков:

Игроки Дополнительный коэффициент
≤ 16 -0.2
≥ 24 0
≥ 40 0.1
≥ 80 0.2

Количество игр:

Игры Дополнительный коэффициент
≤ 4 -0.2
≥ 8 0
≥ 10 0.1
≥ 12 0.3

Например, коэффициент турнира с 10 игровыми сессиями и 60 игроками: coefficient = 1 (base) + 0.1 (sessions) + 0.1 (player) = 1.2
